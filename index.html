О курсе

Этот курс предназначен для тех, кто хочет начать с нуля. В рамках курса вы изучите основы HTML, CSS, JavaScript, а также получите навыки работы с современными инструментами разработчика.

Расписание занятий

  • Занятие 1: Введение в веб-разработку
  • Занятие 2: Основы HTML
  • Занятие 3: Основы CSS
  • Занятие 4: Основы JavaScript
